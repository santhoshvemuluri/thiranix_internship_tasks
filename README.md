# Customer Segmentation Project

## Overview
This project segments a customer base using behavioral and demographic data to identify distinct purchasing patterns. By grouping customers into distinct clusters, businesses can tailor their marketing strategies, optimize product offerings, and gain actionable targeted insights.

## Key Features
* **Data Clustering:** Utilizes Python and `scikit-learn`'s K-Means clustering algorithm to group customers.
* **Behavioral Analysis:** Analyzes patterns based on Annual Income and Spending Scores to determine customer preferences.
* **Visualization:** Employs `matplotlib` and `seaborn` to create clear, 2D visual representations of the customer segments and their core characteristics.

## Expected Outcome
* Successfully categorized customers into 5 distinct segments.
* Gained practical experience in customer analytics and machine learning deployment.
* Generated targeted insights that can be used for personalized marketing campaigns.

## Technologies Used
* Python 3.x
* Pandas & NumPy (Data manipulation)
* Scikit-Learn (K-Means Clustering & Scaling)
* Matplotlib & Seaborn (Data Visualization)
